# 期末冲刺台

基于期末考试复习范围、真题内容和两套听力 MP3 制作的本地优先复习网站。

## 功能

- 按 2026 年 6 月 25 日考试日期生成每日复习任务
- 听力、翻译、词汇、阅读、写作五类专项练习
- 练习模式即时判分和解析
- 75 题、100 分、100 分钟结构的模拟考试
- 错题、重点标记、复习进度和浏览器本地保存
- 桌面与手机响应式布局

## 本地开发

```powershell
npm install
npm run dev
```

浏览器打开终端显示的本地地址。

## 生产构建

```powershell
npm run build
```

构建结果位于 `dist/`。部署时上传 `dist/` 内的全部文件即可。站点使用 Hash 路由，适用于普通静态文件托管。

## 数据说明

学习进度保存在当前浏览器的 `localStorage` 中，不会上传到云端。清理浏览器站点数据会同时清除进度。
